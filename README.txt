This is a Unity program that will generate musical rythyms and the sheet music for them.
This can be useful for teaching kids how to clap rythyms or for using as rythym sightreading
To get started, Go into the 'Build' Folder and double click the 'Rythym Generator' application.
On the main menu of the application there is a button to take you to a guide for using this tool effectively
Feel free to do whatever you want with this project!
